# web_ui

Здесь будет жить общий web-интерфейс `Smart Platform`.

## Цель

Один и тот же визуальный язык для:

- `ESP32`
- `Raspberry Pi`

## Что сюда войдет позже

- общий shell;
- страницы модулей;
- сервисные страницы;
- компоненты статуса и блокировок;
- единые стили и иконки;
- словарь визуальных состояний.

## Важное правило

UI должен быть одинаковым по внешнему виду, но не обязан предоставлять одинаковые полномочия.

Например:

- на `ESP32` страница турели может быть видна, но заблокирована;
- на `Raspberry Pi` эта же страница будет активной.
