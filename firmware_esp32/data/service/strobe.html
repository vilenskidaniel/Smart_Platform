Strobe Bench

Лабораторный экран для стендовой проверки стробоскопа на ESP32. Здесь мы отдельно тестируем локальный bench-контур, не подменяя owner-side боевой strobe турели.

Управляющие команды остаются защищенными service-mode. Вкладки ниже дают быстрый доступ к ручному pulse, burst, loop, timed continuous-on, безопасным пресетам и живому техническому статусу.

Strobe Lab Tabs

Одна страница без reload, переключение происходит внутри лабораторного экрана.
Сценарий Этот экран предназначен для последовательной лабораторной проверки канала: сначала service-mode и arm, затем короткий pulse, затем burst, затем loop или timed continuous, и только потом повторяемый preset.

Следующий шаг для сервиса: сохранить подобранные bench-параметры в общий профиль, чтобы потом их можно было применять из `manual FPV` и `automatic`.

Manual Pulse

Короткая одиночная вспышка для безопасной проверки канала.

Burst Pattern

Серия вспышек с отдельной настройкой количества, on и off таймингов.

Loop Pattern

Повторяющийся цикл on/off с внутренним runtime limit и safety timing checks.

Timed Continuous On

Непрерывное включение с обязательным timeout, чтобы bench не остался включенным бесконтрольно.

Safe Presets

Готовые безопасные bench-профили для повторяемой проверки.
